# g2i-trivia-app

Built June 10, 2020 by Gregory Padron.

### Scripts

`npm start`: Start the Webpack Dev Server

`npm run start:prod`: Build for Production and Start Express

`npm run lint`: Lint

`npm run prettier`: Prettier

`npm run build`: Build for production

### Tech stack:

- React 16.4.1
- Redux 4.0.0
- Express 4.16.3
- Babel 6.26.3
- Webpack 4.16.2
- ESLint 7.2.0
- Prettier 1.13.7
- React-Router 4.3.1
